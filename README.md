# vsventures.org

Single-file static website for **VSv Inc.** — a boutique M&A advisory firm in the Philippines specializing in BPO, CX, and tech-enabled services.

## Tech
- Pure HTML / CSS / JS (no build step)
- Hosted on Netlify
- Auto-deploys from this `main` branch on push
- Domain: vsventures.org (DNS managed at GoDaddy, pointed to Netlify)

## Editing
Just edit `index.html` directly on the `main` branch. Netlify rebuilds and ships in ~30 seconds.

## Layout
- `index.html` — full site (CSS + JS embedded)
- `assets/` — logos, leadership photos, downloadable company profile PDF
